# Contacts Keeper Application
NodeJS, Express Server and with ReactJS
Demo Application [Contact Keeper](https://pacific-shelf-03384.herokuapp.com/).
### Version 1.0.0
Add contacts with
  - Person Name
  - Email address
  - Phone Number and
  - Contact type
# Getting Start
```
npm install
```
# Client Install
```
npm install clientinstall
```
